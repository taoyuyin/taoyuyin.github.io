利用 GitHub 和 Hexo 构建博客

创建github仓库

创建一个以 owner.github.io 命名的仓库，可以直接生成 xxx.github.io 域名的GitHub Pages，推荐用这种命名方式生成博客。

图例

我的以 taoyuyin.github.io 命名，创建完成就可以直接访问了，https://taoyuyin.github.io/

发布
